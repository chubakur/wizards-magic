# Versions #

e.g. 1.0.1
MAJOR.MINOR.CONTENT

**MAJOR** - major changes

**MINOR** - minor changes

**CONTENT** - added items ( not engine-source ). e.g. card, card`s image, sound, etc...

# Running #

For running this game you need installed [Python2](http://python.org/) and [Pygame](http://pygame.org).
